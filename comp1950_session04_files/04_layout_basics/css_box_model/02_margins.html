CSS box model: margins

<body> Margin

The <body> tag atually has margins of its own. If left to the default, this will cause a space between your page content and the edges of the viewport. Note how this page content is contained by #wrapper, which is colored light yellow. Also note how the white color of the viewport is visible above and at the sides of #wrapper.

Apply margin:0px; to the <body> to remove this space.
