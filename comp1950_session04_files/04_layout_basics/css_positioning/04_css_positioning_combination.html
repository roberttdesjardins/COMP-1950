CSS Positioning: Combining Absolute and Relative

There are countless ways to combine both relative and absolute positioning. Heres one useful technique: Since absolute positioned elements are placed based the edges of the available space, if absolute positioned box A is contained in a relative positioned box B, then A is absolute positioned based on Bs edges.
