Absolute positioning places the box model according to the edges of the available space. The available space is determined by the first ancestor element with a position other than static. If no such element exists then the available space is determined by the containing <html> tag.

When absolute positioning, at least two coordinates are usually described, an X axis (right or left) and a Y axis (top or bottom).

  • begin with position:absolute;
  • then assign one or two of the coordinates: top, right, bottom or left
  • use with block elements
  • often used with divs
  • applied using ids (unique on the page)
  • removes the element from normal flow: it no longer affects other element positions

Absolute positioning places the box model according to the edges of the available space. The available space is determined by the parent element.

When an element is a child of the body, absolute positioning coordinates describe how far from the edges of the browser the box model should appear. When absolute positioning, at least two coordinates are usually described, an X axis (right or left) and a Y axis (top or bottom).
