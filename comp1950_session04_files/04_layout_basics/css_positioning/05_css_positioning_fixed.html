Fixed positioned is bound to the browser viewport. Fixed positioning removes the element from normal flow.

CSS Positioning: Fixed

Similar to absolute positioning, except fixed content will NOT scroll with the page

  • begin with position:fixed;
  • then assign one or two of the coordinates: top, right, bottom or left
  • coordinates will always apply to the viewport
  • use with block elements
  • often applied with classes and ids
  • removed from static flow

Examine the CSS of this page to see how the fixed position is applied.
