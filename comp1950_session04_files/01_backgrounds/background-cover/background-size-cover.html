Using background-size:cover;

Cover any sized container with a background image that will not be skewed using the following CSS properties:

  • background-image: url("image.jpg"); load the image (if the container is large, be sure to use a good resolution image!)
  • background-attachment: fixed; fix the position so it wont move with the scroll
  • background-size:cover; let CSS cover the container with the image without skewing
