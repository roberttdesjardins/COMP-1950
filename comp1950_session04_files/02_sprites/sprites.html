Sprites

Using one larger image instead of several smaller images can result in a more effiecient server/client transaction, since it reduces the number of request/responses that need to occur. See the source code for details...

  1. use an image program like photoshop to create a strip or grid of images. this is also an ideal time to determine the correct offsets
  2. use an HTML tag ('div' or 'p' or etc) any block element
  3. define a fixed height and width (large enough to show the image you need to display)
  4. use the url and position offset values for example:
    • background: url("myimage.jpg") 0 0;
      places myimage.jpg in the top left corner ( 0 0 )
    • background: url("myimage.jpg") -50 0;
      places myimage.jpg at the top, but shifts the start of the image 50 pixels to the right (offset -50px)
    • background: url("myimage.jpg") 50 0;
      places myimage.jpg at the top, but shifts the start of the image 50 pixels to the left (offset -50px)
