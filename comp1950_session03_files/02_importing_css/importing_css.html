Importing CSS

The @import Method

You can use the @import method to import multiple stylesheets from a basic stylesheet that you link to. This will reduce the clutter that can sometimes occur within the <head> section. It also increases site maintainability, since adding new stylesheets to your website pages no longer requires changing the head section of every html page.

It is a great place to be able to manage your CSS from

Example:

From your main document, create a <link> to your styles.css page.

In the body of the styles.css file, import as many files as you need:

  • @import url("/css/type.css");
  • @import url("/css/nav.css");
  • @import url("/css/template.css");
